# Proyecto Final (Python Bootcamp)
Este proyecto es una simulación básica de un torneo, desarrollado en Python como parte de una tarea de un bootcamp de Introducción a Ciencia de Datos, Python e Inteligencia Artifical. El objetivo del proyecto es aplicar fundamentos de programación, refrozando el aprendizaje inicial en Python.

## Tecnologías Utilizadas
- Python
- Google Colab (Jupyter Notebook)
